*Freelance project management with client communication, progress tracking, invoicing, and contract management.*

A comprehensive project management assistant built specifically for freelancers and independent consultants. Handles project planning, client communication, milestone tracking, invoicing, and contract management so you can focus on delivering great work.

## Features

| Feature | Description |
|---------|-------------|
| Project Planner | Creates detailed project plans with milestones, deliverables, and timeline estimates |
| Client Communication | Drafts professional updates, status reports, and scope change notifications |
| Progress Tracker | Monitors task completion, budget burn rate, and deadline adherence |
| Invoice Generator | Creates professional invoices with time tracking, expenses, and payment terms |
| Scope Manager | Detects scope creep, documents change requests, and manages client expectations |
| Contract Assistant | Drafts and reviews freelance contracts, SOWs, and NDAs |

## Getting Started

```bash
# Install via OpenClaw
openclaw install AIibaba/openclaw-ai-freelance-pm-skill
```

## Usage

```
plan project [brief]       # Create a comprehensive project plan
status update [project]    # Generate client-ready status report
track [project]            # Review progress and flag risks
invoice [project/period]   # Generate a professional invoice
scope check [request]      # Evaluate if a request is in scope
```
